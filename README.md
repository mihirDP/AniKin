## 🌟 What is AniKin?

AniKin is a highly polished, modular, and professional-grade dockable toolbar designed directly for character animators in Maya. It brings the power of industry-standard tools (like advanced tweening, smart baking, timeline range operations, and pose caching) directly to your viewport with a clean, flat, modern UI. 

Whether you're animating keyframe-by-keyframe, cleaning up dense motion capture, or exporting to Unreal Engine, AniKin removes the technical friction so you can focus on the art of animation.

## ✨ What's New in v0.5.0

- **Complete UI Redesign** — New 10-group toolbar layout with clear visual grouping and separators
- **New Icon Pack** — Custom-designed V2 SVG icons for every tool, replacing the old Tabler-based icons
- **Amber Accent Theme** — New dark-mode color system with amber (`#d4860a`) as the primary accent
- **Merged Lock/Unlock** — Single toggle button replaces the old two-button lock/unlock pair
- **Brand Identity** — Clickable "AniKin" logo in the toolbar opens About/Settings
- **Reorganized Layout** — Tools grouped logically: Brand → Selection → Pose → Tween → Tangents → Playback → Channels → Modules → Curves → Setup
- **Compact Buttons** — Smaller 28×28px buttons (down from 36×36) for a tighter, more professional toolbar
- **Danger Styling** — Delete Key button now shows red on hover for safety feedback

AniKin is divided into specialized modules, accessible right above your timeline. The toolbar is **fully customizable** — you can hide or reorder sections in the Settings panel!

## ⚙️ Installation

1. Download or clone this repository to your machine.
2. Drag and drop the `install.mel` file into your Maya viewport.
3. The AniKin toolbar will dock automatically to the bottom of your workspace!

For a full guide on every feature, see the [Tutorial](TUTORIAL.md).
