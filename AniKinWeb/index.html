Introduction to AniKin

Welcome to AniKin, an open-source animation productivity and kinematics diagnostic toolkit for Autodesk Maya.

AniKin is built with one central mission: to merge daily productivity utilities with robust physics-based kinematics diagnostics. v0.2.0 ships the core productivity module layout, while future releases will introduce automated animation check engines.

Note: AniKin is 100% free and open-source under the GNU GPLv3 license.

Core Philosophy

Unlike plugins that pack loose scripts together, AniKin organizes every tool as a clean package namespace. This allows technical directors and animators to toggle individual features without bloating their setups.

Installation Guide

AniKin can be installed either via standard drag-and-drop or manually inside your Maya environment path.

Option 1: Drag-and-Drop (Recommended)

This is the fastest method and automatically hooks the setup into your Maya session shelf:

  1. Download the latest release archive.
  2. Extract the files to a safe location on your computer.
  3. Drag the file drag_drop_install.py from the extracted folder and drop it into Maya's 3D viewport.
  4. The AniKin toolbar will launch immediately.

Option 2: Manual Module Setup

If you prefer a manual setup (e.g. for pipeline script distribution):

  1. Copy the AniKin directory to your Maya modules folder.
  2. Copy anikin.mod to the same directory.
  3. In Maya's Python Script Editor, run:
import anikin
anikin.launch()

AniTween (Tween Slider)

The AniTween module allows animators to quickly calculate breakdowns and blended keyframes relative to the surrounding keyframes.

Key Features

  • Linear Interpolation (TW): Traditional blending slider.
  • Ease Interpolation (EA): Easing logic (Ease-In / Ease-Out) built using smoothstep and curve acceleration.
  • Overshoot: Dragging sliders beyond 100% or below 0% inserts overshoot values, ideal for quick weight adjustments.

AniOffset (Stagger)

The AniOffset module is used to stagger keyframes across multiple selected controls.

Usage

Select multiple controls in sequence (e.g., tail segments, joints), and click the Offset button on the toolbar. The keys of each subsequent control will offset by a set amount of frames, creating natural overlap and follow-through actions instantly.

AniBake (Smart Bake)

The AniBake module handles baking translations and rotations in world space to a temporary locator, allowing you to tweak keyframes or constraints, and bake the modified offsets back onto the target controls.

Locator Round-trip

  1. Select your control node.
  2. Click the Bake to Locator button. A temporary locator is created matching the node's world space.
  3. Perform changes (adjust pivots, parent constraints, offset curves).
  4. Select the target control node and click Bake from Locator. The animations are baked back onto the target.

AniMirror (Pose Tools)

AniMirror handles copying, pasting, and mirroring poses across asymmetrical character controls.

Mirror Pose

Uses axis negation rules to translate pose vectors across symmetry. Ideal for cycling poses or correcting asymmetric joint configurations during cycles.

AniGhost & AniMotion

These two modules wrap Maya's native viewport visualization tools inside simple, fast buttons.

  • AniMotion (Motion Trails): Creates or removes interactive motion paths for selected controllers.
  • AniGhost (Ghosting): Displays frame overlays (pre- and post-frames) in the viewport to inspect key poses.

Hotkey Setup

AniKin contains a built-in Hotkey Manager to record and bind shortcuts to any tool in the toolbar.

Click the Keyboard icon on the toolbar to open the manager. Press your desired keys, and click Apply. AniKin binds these actions natively inside Maya's runtime commands, ensuring they persist after restarts.

Reordering Toolbar

You can reorganize your toolbar sections or hide tools you don't use often.

Click the Settings Gear icon on the toolbar, and navigate to the "Toolbar Layout" tab. Use the checklist to toggle visibility, and drag or use arrow keys to sort the sequence of your tool groups.
