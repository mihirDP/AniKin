# AniKin Icons

This directory is intended for **original** icon artwork only. 
Do not place copied or trademarked icons in this directory.

AniKin automatically detects `.png` or `.svg` files placed here and maps them to toolbar buttons based on the button's internal name.

## Naming Convention
The UI automatically replaces spaces with underscores and makes names lowercase.
Supported mapping strings include:
- `align_all.svg` (Align)
- `align_translate.svg` (AlignT)
- `align_rotate.svg` (AlignR)
- `auto.svg`, `flat.svg`, `linear.svg`, `step.svg`, `spline.svg` (Tangents)
- `nudge_left.svg`, `nudge_right.svg` (Nudge 1 frame)
- `nudge_left_fast.svg`, `nudge_right_fast.svg` (Nudge 5 frames)
- `offset.svg` (Offset Keys)

- `trail.svg`, `ghost.svg` (Vis)
- `selection_sets.svg` (Sel Sets)

## Specifications
- **Grid:** 24x24px
- **Stroke Weight:** 2px
- **Style:** Clean, geometric line-art (Lucide / Phosphor style reference)
- **Format:** SVG is preferred for crisp scaling across HiDPI monitors.
